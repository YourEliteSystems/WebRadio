# 📻 WebRadio

[![Node.js Version](https://img.shields.io/badge/node-%3E%3D22.14-brightgreen)](https://nodejs.org/) 
[![npm](https://img.shields.io/badge/npm-recommended-blue)](https://www.npmjs.com/) 
[![License](https://img.shields.io/badge/license-MIT-lightgrey)](LICENSE)

**WebRadio** ist eine Electron- und Node.js-basierte Anwendung zum Abspielen von Internet-Radios weltweit. Die App läuft performant auf nahezu jedem Rechner und benötigt keine externe Konfiguration – alle Sender und Funktionen sind direkt im Programm integriert.

---

## 🚀 Features

- Weltweite Radiosender  
- Plattformübergreifend: Windows, macOS, Linux  
- Hochperformant und ressourcenschonend  
- Benutzerfreundliches Interface mit direktem Zugriff auf alle Sender  
- Favoriten und Presets direkt im Programm  
- **Nahtloser Senderwechsel** ohne Unterbrechung  
- **FFMPEG-basiertes Backend**, dauerhaft überwacht und bei Problemen automatisch neugestartet  
- Erweiterbares Plugin-System  
- Unterstützung für individuelle Themes
